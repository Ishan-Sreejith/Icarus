## FIXES APPLIED THIS SESSION

### Fix 1: is_map Builtin Function Support

**Problem:**
- IR generator creates `is_map` function calls in for loops
- JIT compiler didn't recognize is_map as a builtin
- Error: "Unknown function: is_map"

**Solution:**
- Added handlers in src/jit/compiler.rs (lines 467-484):
  ```rust
  } else if func == "is_map" {
      // Return 0 (not a map) since maps not supported yet
      if let Some(d) = dest {
          let r = self.regmap.alloc(d)?;
          self.locals.insert(d.clone());
          emit.emit_mov_imm(r, 0);
      }
  } else if func == "is_list" {
      // Return 0 (not a list) since lists not supported yet
      ...
  } else if func == "is_string" {
      // Return 0 (not a string) since strings not supported yet
      ...
  }
  ```
